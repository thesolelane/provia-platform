# Massachusetts Stretch Code — Preferred Builders Reference
# Applies to all projects in stretch code municipalities
# Last updated: 2024

## WHAT IS THE STRETCH CODE?
The Massachusetts Stretch Code (Appendix 115.AA of the state building code) is an enhanced energy 
efficiency standard adopted voluntarily by municipalities. Once a town adopts it, ALL new residential 
construction and major renovations must comply.

## KEY STRETCH CODE REQUIREMENTS FOR NEW CONSTRUCTION

### INSULATION MINIMUMS
- Attic/ceiling: R-49 minimum (vs R-38 standard)
- Walls: R-20 continuous OR R-13 + R-5 CI (vs R-13 standard)
- Foundation/basement walls: R-15 continuous
- Slab edge: R-10 for 4 feet
- Windows: U-0.30 max, SHGC 0.40 max

### AIR SEALING
- Blower door test required: ≤ 3.0 ACH50 (air changes per hour at 50 pascals)
- Must be tested by certified energy rater BEFORE certificate of occupancy
- Typical cost to test: $300–500

### HVAC
- Heat pumps strongly favored (meet efficiency tiers without fossil fuel exception)
- Gas/oil equipment: must meet minimum AFUE requirements
- ERV (Energy Recovery Ventilation) required when air sealing achieves < 3 ACH50
- Mini-split systems (inverter-driven) automatically satisfy HVAC stretch code requirements

### LIGHTING
- 90% of all permanently installed lighting must be high-efficacy (LED)
- Exterior lighting: must be ENERGY STAR certified

### WATER HEATING
- Water heater efficiency minimums elevated
- Heat pump water heaters satisfy all requirements
- If gas: 0.67 EF minimum

### SOLAR READY (if applicable)
- New homes over 1,200 sq ft must be "solar ready" (conduit from electrical panel to roof, 
  marked location for future panels)

## HOW THIS AFFECTS PREFERRED BUILDERS PROJECTS

### Always Include in Stretch Code Towns:
1. Blower door test (add to permit cost line — ~$400)
2. Energy rater certification (sometimes included with blower door)
3. Upgraded insulation per above minimums
4. ERV unit if tight house (< 3 ACH50 likely) — typically $1,500–2,500 installed
5. Permit line item notation: "Stretch Code Compliance"

### Mini-Split Systems (like Jones Hill Rd):
✓ SATISFY Stretch Code HVAC requirements — excellent choice
✓ ERV recommended alongside tight building envelope
✓ No gas fossil fuel exception issues

### Spray Foam vs. Batts:
- Closed-cell spray foam: meets wall R-value with less thickness
- Open-cell: good for attics, meets R-49 at appropriate thickness
- Standard fiberglass batts: require continuous insulation (CI) layer to meet wall requirements

## PERMIT & INSPECTION CHECKLIST FOR STRETCH CODE TOWNS

- [ ] Submit energy compliance form with permit application
- [ ] Insulation installed per specs — inspected BEFORE drywall
- [ ] Air sealing completed — rough-in inspection
- [ ] Blower door test scheduled (before CO)
- [ ] HVAC equipment specs submitted (must meet minimums)
- [ ] Lighting plan reviewed
- [ ] ERV installed and balanced (if applicable)
- [ ] Final CO from building department after passing blower door

## COST IMPACT ON ESTIMATES

Add to permit/compliance line for stretch code towns:
- Blower door test: $350–500
- Energy rater fee (if separate): $200–400
- ERV unit + installation: $1,500–2,800
- Upgraded insulation (vs standard): $800–2,500 depending on scope
- TOTAL TYPICAL ADDER: $2,850–6,200

For Jones Hill Rd (Ashby): $10,350 permit line covers all stretch code compliance.
